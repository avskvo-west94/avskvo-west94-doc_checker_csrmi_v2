# Инструкция по установке и первому запуску

## Требования

- Windows 11 (x64)
- .NET 8.0 Runtime или SDK

## Установка .NET 8.0

Если .NET 8.0 еще не установлен:

1. Перейдите на https://dotnet.microsoft.com/download/dotnet/8.0
2. Скачайте и установите .NET 8.0 SDK (для разработки) или Runtime (для запуска готового приложения)

## Сборка проекта

### Вариант 1: Из исходного кода

```bash
# Клонируйте репозиторий
git clone https://github.com/avskvo-west94/avskvo-west94-doc_checker_csrmi_v2.git
cd avskvo-west94-doc_checker_csrmi_v2

# Восстановите зависимости
dotnet restore

# Соберите проект в режиме Release
dotnet build --configuration Release

# Запустите приложение
dotnet run --configuration Release
```

### Вариант 2: Готовый исполняемый файл

После сборки исполняемый файл будет находиться в:
```
bin/Release/net8.0-windows/DocumentChecker.exe
```

Вы можете скопировать эту папку на любой компьютер с Windows 11 и .NET 8.0 Runtime и запустить приложение.

## Первый запуск

1. **Запустите приложение** `DocumentChecker.exe`

2. **Создайте первый шаблон:**
   - Меню → **Шаблоны** → **Управление шаблонами**
   - Нажмите **Создать**
   - Введите название шаблона (например, "Шаблон для проверки изделий")
   - Заполните эталонные значения для 11 обязательных полей:
     - Наименование изделия
     - Модель/модификация
     - Наименование производителя
     - Страна производителя
     - Наименование заявителя
     - Страна заявителя
     - Адрес производителя
     - Адрес заявителя
     - Наименование испытательной лаборатории
     - Регистрационный номер/код изделия
     - Дополнительное поле
   - Нажмите **Сохранить**

3. **Выберите шаблон:**
   - Меню → **Шаблоны** → **Выбрать шаблон**
   - Выберите созданный шаблон и нажмите **Выбрать**

4. **Проверьте документ:**
   - Меню → **Файл** → **Открыть документ**
   - Выберите Word документ (.docx)
   - Нажмите кнопку **Проверить документ**
   - Просмотрите результаты в таблице справа

5. **Исправьте несоответствия:**
   - Выберите несоответствия для исправления (чекбокс "Исправить")
   - Нажмите **Исправить выбранное**
   - Сохраните исправленный документ: **Файл** → **Сохранить исправленный документ как...**

## Где хранятся данные

Шаблоны сохраняются автоматически в:
```
%APPDATA%\DocumentChecker\templates.json
```

Этот файл можно скопировать на другой компьютер для переноса шаблонов.

## Устранение проблем

### Ошибка "dotnet: command not found"
- Установите .NET 8.0 SDK с официального сайта Microsoft

### Ошибка при открытии документа
- Убедитесь, что файл имеет расширение `.docx` (не `.doc`)
- Убедитесь, что файл не открыт в другой программе
- Проверьте права доступа к файлу

### Приложение не запускается
- Убедитесь, что установлен .NET 8.0 Runtime
- Проверьте логи ошибок в папке приложения
